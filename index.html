How to use FretLocal

  • Pick strings — choose which guitar strings to practice (1–6).
  • Set reps — how many notes to be tested on.
  • Exercise — a note appears. Click the fret where it lives on each selected string, then hit Submit.
  • Review — green = correct, red = wrong click, orange = missed the right fret.

Frets shown: 1–12 (standard tuning). Each note appears exactly once per string in this range.

Standard tuning: E₂ · A₂ · D₃ · G₃ · B₃ · E₄ (low to high)
