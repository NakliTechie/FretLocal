How to use FretLocal

  • Pick a preset — load a famous riff and drill those exact notes in order.
  • Or go custom — choose any strings and set how many random reps.
  • Exercise — a note appears; click the fret where it lives on each selected string, then hit Submit.
  • Review — green = hit · red = wrong click · orange = missed the correct fret.

Sound: Karplus-Strong plucked string synthesis. Each click plays the real pitch of that fret. Use 🔊 to mute.

Frets 1–12 shown. Each of the 12 chromatic notes appears exactly once per string in this range (standard tuning: E₂ · A₂ · D₃ · G₃ · B₃ · E₄).
